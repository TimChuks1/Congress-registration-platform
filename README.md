# Conference Management Portal

Congress management portal for attendee registration, conference information, and on-site registration verification.

## Features

- Conference registration
- Conference information
- Speakers
- Seminar classes
- Bible Study
- Hymnal
- Program schedule
- FAQ
- Registration verification/check-in
- Admin dashboard

## Tech Stack

### Frontend

- React
- Vite
- Tailwind CSS

### Backend

- Node.js
- Express.js
- MongoDB
- JWT Authentication
